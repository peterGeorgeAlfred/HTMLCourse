Lecture 1: Web Development Fundamentals

1. Introduction to Websites

  • Websites can be hosted by individuals, businesses, or organizations to share information.
  • Some websites are free, while others require subscriptions.
  • Categories of Websites:
    • Personal Websites: Blogs, portfolios, and personal projects.
    • Commercial Websites: E-commerce sites and online stores.
    • Organizational Websites: Government, educational, and non-profit organizations.
    • Entertainment Websites: Streaming services, gaming sites, and news portals.

2. Components of a Website

A Web Server hosts the website and delivers web pages using the HTTP protocol. It can be:

  • Intranet Server: Used internally within an organization.
  • Internet Server: Accessible publicly.

3. Static vs. Dynamic Web Pages

  • Static Web Pages: Delivered directly from the server without modification. They display the same content to all users.
  • Dynamic Web Pages: Generated by web applications or databases before being sent to the user. They can display different content based on user interactions or other factors.

4. HTML Basics

HTML (Hyper Text Markup Language) is the standard language for creating web pages. It consists of a series of elements that define the structure of the page.

Example: Simple HTML Document

<!DOCTYPE html>
<html>
<head>
  <title>Page Title</title>
</head>
<body>
  <h1>My First Heading</h1>
  <p>My first paragraph.</p>
</body>
</html>

5. HTML Elements and Attributes

  • Headings (<h1>, <h2>, etc.) define the title structure.
  • Attributes provide extra information for elements, such as:
    • href: Specifies the URL for hyperlinks.
    • src: Specifies the image source.
    • alt: Alternate text for images.
    • style: Inline CSS for styling elements.

6. Styling with CSS

CSS (Cascading Style Sheets) is used for styling HTML elements.

Example: Changing Background and Text Colors

<h1 style="color:blue;">This is a heading</h1>
<p style="background-color:tomato;">This is a paragraph.</p>

7. HTML Text Formatting

  • Bold Text: <b>
  • Italic Text: <i>
  • Underlined Text: <u>
  • Highlighted Text: <mark>

8. Working with HTML Colors

Colors can be defined using:

  • Names (e.g., "Tomato")
  • RGB (e.g., rgb(255, 0, 0))
  • HEX values (e.g., #FF0000)

Example: Using a Color in HTML

<h1 style="background-color:DodgerBlue;">Hello World</h1>
